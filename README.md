## Employee Attrition Analysis

### Tools Used
- Python (Pandas, Matplotlib)
- VS Code
- Tableau

### Project Workflow
1. Data cleaning and preprocessing using Python scripts
2. Exploratory Data Analysis (EDA)
3. Feature engineering (Age Groups, Salary Slabs)
4. Exported data for Tableau
5. Built interactive dashboard in Tableau Public

### Key Insights
- Sales department shows highest attrition
- Low salary slab employees have higher attrition
- Overtime significantly increases attrition
- Younger employees show higher attrition
